OpenCode Tailnet Launcher v0.01

Single-exe portable release.

1. Run OpenCodeTailnetLauncher.exe.
2. It stays in the tray and silently keeps opencode web alive.
3. It never opens a browser by itself.
4. Double-click the tray icon to open the router page.
5. If no config file exists, it creates oc-launcher.ini automatically beside the exe.

Commands:
- OpenCodeTailnetLauncher.exe --install-autostart
- OpenCodeTailnetLauncher.exe --remove-autostart

Generated files after first run:
- .\oc-launcher.ini
- .\logs\launcher.log
